#
# 恋爱通过 安装说明.txt
#

1. 安全起见, 请备份站点的所有文件和数据库
2. 解压文件并上传至服务器文件的apps目录中
3. 进入网站后台安装此应用
4. 注册一个应用的官方微博账号，昵称最好能和应用相呼应
3. 进入"管理后台 - 应用 - 应用配置 - 恋爱通告", 进行相关配置（注意：官方微博账号的设置是ID，非用户名或者邮箱）
5. 清理缓存
4. Enjoy

+ 其他说明
  - 开启伪静态和个性化域名:  参见"开启URL伪静态的方法.txt"
  - 恢复或删除漫游平台信息:  参见"恢复或删除漫游平台站点信息的方法.txt"
  - ThinkSNS集成UCenter: 参见"集成UCenter的方法.txt"
